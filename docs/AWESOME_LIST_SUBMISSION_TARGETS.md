# Awesome List Submission Targets

Prioritize relevant, maintained lists and communities. Submit only where the repo clearly fits.

## Highest-priority targets

- Awesome music production lists
- Awesome audio/DSP lists
- Awesome WebAudio lists, if they include platform/resources sections
- Awesome audio visualization lists, if submitting the visualization subset
- Independent music/producer resource directories
- Open-source audio communities
- Music-tech newsletters and GitHub roundups

## PR rule

Do not spam. Add this repo only to categories where it is genuinely useful, such as:

- Music resources
- Music platforms
- Artist tools
- Audio resources
- Music marketing
- Music distribution
- Creator tools

## Backlink network rule

When accepted by another list, add the source to a future `docs/BACKLINK_LOG.md` file with:

- date
- source repo/site
- accepted URL
- category
- notes
