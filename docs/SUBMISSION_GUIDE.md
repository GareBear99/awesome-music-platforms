# Submission Guide

Use this guide when adding platforms to the list.

## Entry style

Use this format:

```markdown
- [Name](https://example.com/) - Short neutral description.
```

## Good description examples

- Music distribution service for releasing tracks to streaming platforms.
- Beat marketplace for producers and artists.
- Smart-link platform for music marketing and pre-save campaigns.
- Open-source audio editor for recording and editing sound.

## Bad description examples

- The best platform ever.
- Guaranteed to make you viral.
- Sign up with my link.
- Cheap followers and streams.

## Category rules

Use the most specific category available. If a platform fits multiple categories, choose the category where users would most likely search for it.
