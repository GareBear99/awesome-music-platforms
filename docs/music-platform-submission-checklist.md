# Music Platform Submission Checklist

Use this before submitting a new platform to this list or before uploading your own release to a public platform.

## Platform entry checklist

- Public URL works
- Platform is relevant to artists, producers, labels, listeners, developers, or music-tech creators
- Description is short and neutral
- No spam, scams, malware, or misleading claims
- Category fits the README structure
- Not a duplicate unless it belongs in multiple categories with a clear reason

## Artist release checklist

- Final master or preview audio
- Correct title and artist name
- Cover art at required size
- Genre and mood tags
- UPC/ISRC if distributed
- Lyrics if relevant
- Clean/explicit flag
- Smart link or canonical hub link
- Social caption and short clip

## Repo maintenance checklist

Run link checks, review platform descriptions, update changelog, and keep hub backlinks current.
