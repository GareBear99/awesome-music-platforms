# Where to Sell Loops, Stems, and Sample Packs

Sample-pack creators should use both marketplace reach and direct-store ownership.

## Marketplace reach

- Splice Sounds
- Loopmasters
- ADSR Sounds
- Producer Loops
- Loopcloud
- Cymatics
- LANDR Samples

## Direct-store control

- Gumroad
- Payhip
- Shopify
- WooCommerce
- Bandcamp, for album-like packs and audio bundles

## Packaging checklist

- Clear license text
- BPM and key labels where relevant
- WAV files for audio loops
- MIDI files when useful
- Demo track
- Cover image
- Readme with usage terms
- Folder structure by BPM/key/type
